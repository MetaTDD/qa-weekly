QA周刊第四期—​测试人员和法医

测试和法医

最近分别看完了书籍 《非自然死亡》和日剧 《非自然死亡》，抛开剧情，我觉得测试人员和法医还是很像的。法医为死者言，测试为bug言。测试人员也像法医一样需要经过一系列的“解剖”来定位软件的缺陷甚至“死因”，最终查找出杀死软件的"元凶"。

该表格支持各类 Python 公式与 Excel 函数，拥有可扩展移植的数据格式、轻便的数据编辑功能，并自带可视化数据面板，开发者还可通过 API 与其它数据源对接。
